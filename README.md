A one-page personal portfolio website with sections for current work, about, skills, current learning, GitHub, and contact.

Open `index.html` in a browser to view it.

- Your contact email is set in the `<body>` tag as `sanjaymaharjan555@gmail.com`.

- Your live website link is set to `https://sones28.github.io/`.

- Update the Currently Learning section as your next goals change.
- Replace `assets/portfolio-hero.png` with your own photo or workspace image when you have one.

## Upload To GitHub Pages

For your GitHub account, create a public repository named:

```text
sones28.github.io
```

Upload these files and folders to the repository root:

```text
index.html
styles.css
script.js
README.md
assets/
```

Do not upload `.DS_Store`.

The hidden files `.nojekyll` and `.gitignore` are optional. If your file picker does not show hidden files, you can skip them.

After uploading, go to the repository's Settings, open Pages, choose "Deploy from a branch", then select the `main` branch and `/ (root)`.

Your site should publish at:

```text
https://sones28.github.io/
```
